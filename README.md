## 📂 Projects Overview

This repository contains multiple distinct data analytics projects, each housed in its own directory with its own specific dataset, analysis notebooks, SQL queries, and dashboards.

### 🛒 [1. Customer Trends Data Analysis](./customer_trends_data_analysis)
This project explores **customer shopping behavior** to uncover insights about purchasing patterns, product preferences, and key sales drivers.
- **Tools Used:** Python (Pandas, NumPy, Matplotlib, Seaborn), MySQL, Power BI.
- **Highlights:** Data cleaning, exploratory data analysis (EDA), customer segmentation using SQL, and an interactive Power BI dashboard.
- [👉 View Project](./customer_trends_data_analysis)

### 🚚 [2. Supply Chain Data Analysis](./supply_chain_data_analysis)
*Currently under development / planning phase.*

## 🚀 How to Navigate

Each project directory contains:
- **Raw Data:** The datasets used for the analysis.
- **Notebooks:** Jupyter Notebooks (`.ipynb`) containing the data processing and EDA steps.
- **SQL Scripts:** Queries used for specific data extraction and segmentation.
- **Dashboards:** Visualization files (e.g., Power BI `.pbix` files or exported images).
- **Project README:** A detailed README specific to that project's findings and setup instructions.
