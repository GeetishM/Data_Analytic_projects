
# Data_Analytic_projects

This project explores customer shopping behavior to uncover insights about purchasing patterns, product preferences, and key sales drivers. It aims to help businesses make data-driven decisions to enhance marketing strategies, inventory planning, and customer satisfaction.
## Tech Stack
- **Python** – Data cleaning, preprocessing, and exploratory analysis  
- **Jupyter Notebook** – Interactive environment for analysis and visualization  
- **MySQL** – Data storage and querying  
- **Power BI** – Dashboard creation for visual insights 
## Key Features
- Data cleaning and preprocessing using **Pandas** and **NumPy**  
- Exploratory Data Analysis (EDA) to identify sales and demographic trends  
- SQL-based data querying for deeper insights into customer segments  
- Interactive **Power BI Dashboard** for visual storytelling  
- Actionable insights for business and marketing strategy

## Setup Instructions

1. **Clone this repository**

```bash
    git clone https://github.com/GeetishM/Data_Analytic_projects
```
2. **Navigate to the project folder**
 ```bash
    cd customer_trends_data_analysis
 ```
3. **Ensure the dataset is available**
 ```bash
    customer_shopping_behavior.csv
 ```
 (Included in the repository.)
 
4. **Install the required Python libraries**
 ```bash
    pip install pandas numpy matplotlib seaborn mysql-connector-python

 ```
5. **Launch Jupyter Notebook**
 ```bash
    jupyter notebook
 ```
